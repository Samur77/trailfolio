P2C Communication

Created a child component named "childRelatedContact", similar to the "childAccountComponent." The component uses an API record ID and a @wire adapter to retrieve contact information from the database. The HTML for the component displays contact information, including the account ID and account name, which are retrieved from the "getChildRelatedContacts" method.

Parent-to-child communication in a component involves the parent sending data to the child component. This is achieved by creating a new element in the parent component and passing data to it using the child component's public properties. The child component then displays the data in its HTML.

Child components retrieve data from parent components and display it. To deploy child accounts, navigate to the account page, edit the page, and add the "child account" component. Ensure the property name for child accounts is "childAccounts" and deploy the child component, parent component, and apex controller. 


        import { LightningElement, api, wire } from 'lwc';
        import getChildRelatedContacts from '@salesforce/apex/AccountController.getChildRelatedContacts';
        export default class ChildRelatedContact extends LightningElement {
        @api recordId;
        @wire(getChildRelatedContacts, {childAccId:'$recordId'})
        relatedConList;
        }
